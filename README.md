# PokePP — เกมจับมอนสเตอร์สไตล์ PokeMeow

เว็บเกมจับโปเกมอนเล่นคนเดียว ทำด้วย HTML/CSS/JavaScript ล้วน ไม่ต้องมี backend (มี Cloud Save เสริมได้)
ข้อมูล + สเตตัสจริงดึงจาก [PokeAPI](https://pokeapi.co) สไปรต์ animated จาก
[PokeAPI/sprites](https://github.com/PokeAPI/sprites) ผ่าน jsDelivr CDN เซฟหลักผ่าน `localStorage`

## ไฟล์
| ไฟล์ | หน้าที่ |
|------|---------|
| `index.html` | โครงหน้าเกม |
| `style.css` | สไตล์ (responsive มือถือ/เดสก์ท็อป, รองรับ PWA) |
| `game.js` | ตรรกะเกมทั้งหมด |
| `monsters-data.js` | ข้อมูลมอนสเตอร์ทุก Gen (1–1025 ตัว) — สร้างจาก PokeAPI |
| `cloud.js` | เชื่อม Supabase สำหรับ Cloud Save + Login (ทำงานแบบออฟไลน์ได้ถ้ายังไม่ตั้งค่า) |
| `sw.js` / `manifest.json` / `icon.svg` | Service Worker + PWA (ติดตั้งลงมือถือ เล่นออฟไลน์ได้) |
| `serve.ps1` | static server เล็กๆ ไว้ทดสอบบนเครื่อง (ไม่ต้องใช้ตอน deploy) |
| `build-data.ps1` | สคริปต์ดึงข้อมูลจาก PokeAPI มาสร้าง `monsters-data.js` ใหม่ |
| `CLOUD_SETUP.md` | คู่มือตั้งค่า Cloud Save (Supabase) ทีละขั้น |
| `DEPLOY.md` | คู่มือ deploy ขึ้น GitHub Pages |

## วิธีเล่นพื้นฐาน
- **เลือกเขตในแผนที่** 🗺️ (10 เขต ธีมต่างกัน โปเกมอนต่างกันตามธาตุ) — บางเขตปลดล็อกเมื่อจับครบ N ชนิด แต่ละเขตมีพื้นหลังภาพถ่ายจริงตรงธีม (จาก Wikimedia Commons) + สไปรต์โปเกมอนจริงลอยตกแต่งทับ
- โปเกมอนป่าปรากฏเอง (ทุก 9–16 วิ) พร้อม **เลเวลตามระดับความหายาก** (หายากกว่า = เลเวลสูงกว่า = จับยากกว่า แบบ PokeMeow) และ **อากาศ/กลางวัน-กลางคืน** มีผลต่อธาตุที่ออก
- **เลือกชนิดบอลแล้วแตะเพื่อโยนทันที** — ไม่ต้องพิมพ์ชื่อ ไม่ต้องกดยืนยันซ้ำ
- **ระบบสุ่มความหายาก 2 ชั้นแบบ PokeMeow**: Common 40% · Uncommon 30% · Rare 27% · Super Rare 3% (สุ่มแยก) + Legendary 1/666 (สุ่มแยกอิสระ) + ✨ Shiny 1/8192 (สุ่มแยกอิสระอีกชั้น)
- เจอตัวหายาก/Shiny/อีเวนต์ → **แจ้งเตือนมุมขวาบน**
- จับได้ → เหรียญ + XP ให้ Buddy · โปเกมอนแต่ละตัวมี **Nature/เพศ/IV/มิตรภาพ/ชื่อเล่น** กดดูรายตัวได้ในคลัง

## 🎒 ไอเทมจับ (13 ชนิดบอล + เบอร์รี่)
| บอล | ผล |
|---|---|
| Poké / Premier / Great / Ultra / Luxury | โอกาสจับพื้นฐานไล่ระดับ (Luxury ให้มิตรภาพโบนัสด้วย) |
| Net · Dusk | โบนัสเฉพาะทาง (น้ำ/แมลง · กลางคืน/ถ้ำ) |
| Quick · Timer | โบนัสตามจังหวะ (ปาลูกแรก · ยิ่งขว้างหลายรอบยิ่งแรง) |
| Repeat · Heavy · Beast | โบนัสตามเป้าหมาย (เคยจับแล้ว · Rare/Super Rare · Legendary ล้วนๆ) |
| Master | จับติด 100% |

ร้านค้าขายบอลได้ **ทีละ 1 ชิ้นต่อการซื้อ** (กดซ้ำได้เรื่อยๆ) จับ **ไม่ง่าย** มีโอกาสดิ้นหลุด/หนีทุกครั้ง

**เบอร์รี่** (Razz/Golden Razz) ใช้ได้ตลอดเวลา — เพิ่ม **โอกาสเจอโปเกมอนหายากในสปอว์นถัดไป** (ไม่ใช่โอกาสจับ แบบ PokeMeow จริง)

## ⚔️ ระบบต่อสู้
- **ทีม 6 ตัว + ท่าโจมตีตามธาตุ** สลับตัวกลางสนามได้ · **ตารางธาตุแพ้ทางจริง 18 ธาตุ**
- **สถานะผิดปกติ**: ไหม้ 🔥 / พิษ ☠️ / อัมพาต ⚡ / หลับ 💤 / แช่แข็ง ❄️ (มีภูมิคุ้มกันตามธาตุ)
- **Held Items** (สวมสู้): Life Orb, Choice Band/Specs, Assault Vest, Expert Belt, Leftovers, Focus Sash
- สู้กับโปเกมอนป่าก่อนจับ → ทำ HP ลด → จับง่ายขึ้นมาก (ป่าไม่สลบ เหลือ 1 HP เสมอ)

### 💎 เมก้าอีโวลูชัน + 💥 ไดนาแม็กซ์/G-Max
ข้อมูล/สเตตัส/สไปรต์**จริงจาก PokeAPI** (30 ร่างเมก้า, 20 ร่าง G-Max) — **ไม่มีทางเจอจากการจับป่าปกติ**
ต้อง**ซื้อหินเมก้า/พลังงานไดนาแม็กซ์เองเท่านั้น** (ราคาสูง ให้รู้สึกคุ้มค่าเวลาได้มา) แนบหินให้ตัวที่ตรงสายพันธุ์ในหน้าโปเกมอน แล้วกดใช้ในการต่อสู้ได้ครั้งละ 1 ต่อยก
ต้องมี **กำไลเมก้า/กำไลไดนาแม็กซ์** (ปลดล็อกครั้งเดียวถาวร ซื้อในร้าน) ก่อนถึงจะใช้ได้เลยแม้มีหินแล้ว — ไดนาแม็กซ์ปกติทำได้ทุกตัวที่มีกำไล แต่จะได้ร่าง **G-Max จริง** เฉพาะตัวที่แนบ **หินปลุกพลัง G-Max** ของสายพันธุ์นั้นๆ ไว้ (ซื้อ/แนบแยกต่างหากในหน้าโปเกมอน เหมือนหินเมก้า) ไม่งั้นได้แค่บวก HP/ดาเมจธรรมดา

### 🏆 บอสประจำเขต + ⚔️ ยิม/เดินสายสู้ + 🗼 หอคอยไต่ระดับ
- **บอสประจำเขต**: ท้าดวลตัวแกร่งสุดของเขต ได้ 🏅 เหรียญตรา
- **ยิม 9 ด่าน** (เดินสายแบบ PokeMeow): ทีมศัตรู 3–6 ตัว **คัดจากระดับหายากจริง** (ไม่ใช่สุ่มมั่วๆ) เอซตัวสุดท้ายสเตตัสบูสต์ ปลดล็อกทีละด่าน ได้เหรียญ + 🎖️ Battle Points + **ไอเทมจริง**
- **หอคอยไต่ระดับ** 🗼: ยิ่งสูงยิ่งยาก ของยิ่งดี — **ไม่ฮีล HP ข้ามชั้น** (เสี่ยงจริง) แพ้ = รีเซ็ตกลับชั้น 1 (สถิติสูงสุดเก็บถาวร) ทุก 5 ชั้นเจอบอส ชั้น 12+ อาจเจอบอส G-Max ชั้น 20+ อาจเจอบอสเมก้า

## รันทดสอบบนเครื่อง (Windows)
```powershell
powershell -ExecutionPolicy Bypass -File serve.ps1 -Port 8791
# เปิด http://localhost:8791/
```
> เปิดไฟล์ตรงๆ ด้วย `file://` ก็เล่นได้ แต่ `localStorage`/Service Worker อาจไม่ทำงานเต็มที่ในบางเบราว์เซอร์ — แนะนำให้เสิร์ฟผ่าน http หรือโฮสต์จริง

## ☁️ เปิด Cloud Save (ตัวเลือกเสริม)
ทำตาม [`CLOUD_SETUP.md`](CLOUD_SETUP.md) เพื่อเชื่อม Supabase — ผู้เล่นจะล็อกอิน/สมัครได้ในเมนู ⚙️ แล้วเซฟ sync ข้ามเครื่องอัตโนมัติ ถ้าไม่ตั้งค่า เกมเล่นออฟไลน์ปกติทุกอย่าง (localStorage เหมือนเดิม)
> หมายเหตุ: ตรรกะเกมยังอยู่ฝั่ง client เหมาะกับเล่นแคชวล/กับเพื่อน — ยังไม่มีระบบกันโกงสำหรับการแข่งขันจริงจัง

## ⚠️ หมายเหตุลิขสิทธิ์
ชื่อ/รูป Pokémon เป็นทรัพย์สินทางปัญญาของ Nintendo/Game Freak/The Pokémon Company
เหมาะกับการใช้ส่วนตัว/ทดลอง หากจะเผยแพร่สาธารณะเชิงพาณิชย์ ควรเปลี่ยนไปใช้มอนสเตอร์ต้นฉบับ
โครงสร้าง `monsters-data.js` ออกแบบให้สลับข้อมูล/รูปได้ง่ายในภายหลัง

## เพิ่ม/แก้มอนสเตอร์
แก้ `monsters-data.js` ได้ตรงๆ แต่ละตัวมี:
`id, name, types[], stats{hp,atk,def,spatk,spdef,spd}, rarity, captureRate, genderRate, evolvesTo, evolveLevel, evolveItem`
หรือรัน `build-data.ps1` ใหม่เพื่อดึงจาก PokeAPI (ปรับ `$MaxId` เพื่อเพิ่ม/ลด Gen)

ตารางเมก้า/G-Max (`MEGA_FORMS`/`GMAX_FORMS` ใน `game.js`) เป็นข้อมูล curated แยกต่างหาก อ้างอิง PokeAPI id ของร่างพิเศษโดยตรง — เพิ่มสายพันธุ์ใหม่ได้โดยหา `pokemon-id` ของร่างนั้นจาก PokeAPI แล้วเพิ่มเข้าไปในตาราง
