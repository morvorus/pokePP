# PokePP — เกมจับมอนสเตอร์สไตล์ PokeMeow

เว็บเกมจับโปเกมอนเล่นคนเดียว ทำด้วย HTML/CSS/JavaScript ล้วน ไม่ต้องมี backend
ข้อมูล + สเตตัสจริงดึงจาก [PokeAPI](https://pokeapi.co) รูปเป็นสไปรต์ animated จาก
[PokeAPI/sprites (showdown)](https://github.com/PokeAPI/sprites) เซฟผ่าน `localStorage`

## ไฟล์
| ไฟล์ | หน้าที่ |
|------|---------|
| `index.html` | โครงหน้าเกม |
| `style.css` | สไตล์ (responsive มือถือ/เดสก์ท็อป) |
| `game.js` | ตรรกะเกมทั้งหมด |
| `monsters-data.js` | ข้อมูลมอนสเตอร์ทุก Gen (1–1025 ตัว) — สร้างจาก PokeAPI |
| `serve.ps1` | static server เล็กๆ ไว้ทดสอบบนเครื่อง (ไม่ต้องใช้ตอน deploy) |
| `build-data.ps1` | สคริปต์ดึงข้อมูลจาก PokeAPI มาสร้าง `monsters-data.js` ใหม่ (ในโฟลเดอร์ scratchpad) |

## วิธีเล่น
- **เลือกเขตในแผนที่** 🗺️ (8 เขต ธีมต่างกัน โปเกมอนต่างกันตามธาตุ) — บางเขตปลดล็อกเมื่อจับครบ N ชนิด
- โปเกมอนป่าปรากฏเอง (ทุก 9–16 วิ) พร้อม **เลเวล + ระดับความหายาก** (Common/Uncommon/Rare/Super Rare/Legendary + ✨Shiny)
- **เลือกชนิดบอลแล้ว "ปาบอล"** — ไม่ต้องพิมพ์ชื่อ · บอลดีกว่าโอกาสจับติดสูงกว่า
  - Poké 🔴 (พื้นฐาน) · Great 🔵 (×1.7) · Ultra 🟡 (×2.6) · Master 🟣 (100%)
  - จับ **ไม่ง่าย** มีโอกาสดิ้นหลุด/หนีทุกครั้ง ตัวหายาก/เลเวลสูงยิ่งยาก
- เจอตัวหายาก/Shiny → มี **แจ้งเตือนมุมขวาบน**
- จับได้ → เหรียญ + XP ให้ Buddy · โปเกมอนแต่ละตัวมี **Nature/เพศ/IV/สเตตัสจริง** กดดูรายตัวได้ในคลัง
- **Buddy**: เลือกจากคลัง → รับ XP เลเวลอัพ + **วิวัฒนาการ** ตามเลเวล/หิน
- **ร้านค้า**: ซื้อบอลแต่ละชนิด / เบอร์รี่ / ไข่ / หินวิวัฒนาการ
- **เควสรายวัน** รีเซ็ตทุกวัน · **ไข่** ฟักเมื่อจับครบ 15 ตัว (มีลุ้น Shiny สูงกว่าปกติ)

### ระบบเสริม
- **⚔️ ต่อสู้แบบทีม 6 ตัว + ท่าโจมตี** — จัดทีมสูงสุด 6 ตัว สลับตัวกลางสนามได้ · แต่ละตัวมีท่าตามธาตุ (เลือกโจมตี) · ใช้ **ตารางธาตุแพ้ทางจริง 18 ธาตุ** · สู้ทำ HP โปเกมอนป่าลด → จับง่ายขึ้นมาก (ป่าไม่สลบเหลือ 1 HP)
- **🏆 บอสประจำเขต** — ท้าดวลบอส (ตัวแกร่งสุดของเขต) เพื่อรับ 🏅 เหรียญตรา + รางวัล
- **🍓 เบอร์รี่** — โยนก่อนปาบอลเพิ่มโอกาสจับ (Razz +13% / Golden Razz +32%)
- **👤 โปรไฟล์/สถิติ** — Trainer Level, % เดกซ์, จำนวน shiny/legendary, IV เฉลี่ย, เวลาเล่น, ทีมปัจจุบัน
- **🎁 รางวัลออฟไลน์** — ปิดเกมไปแล้วกลับมามีเหรียญ/บอลสะสมให้ · **📅 Daily login streak** ล็อกอินต่อเนื่องได้โบนัสเพิ่มขึ้น
- **🏆 ความสำเร็จ (Achievements)** — ปลดล็อกอัตโนมัติ ได้เหรียญรางวัล
- **⚙️ ตั้งค่า** — เปิด/ปิดเสียง (มี SFX จริง), ปรับความเร็ว spawn, **Export/Import เซฟ** (สำคัญเวลาฝัง iframe เพราะเซฟผูกกับโดเมน)
- **QOL คลัง** — เรียงตาม IV/จำนวน/ระดับ, ล็อกกันปล่อย, ปล่อยตัวซ้ำ (เก็บ IV สูงสุด)
- **📖 บทเรียนเริ่มต้น** สำหรับผู้เล่นใหม่ + แอนิเมชันจับ 3 จังหวะ + ประกาย Shiny

## 🗺️ เขตในแผนที่
ทุ่งหญ้า🌾(เริ่มต้น) · ป่า🌲 · ทะเล🌊 · ถ้ำ🪨(จับครบ 8) · ภูเขาไฟ🌋(12) · โรงไฟฟ้า⚡(15) · หิมะ❄️(25) · ดินแดนลึกลับ🔮(40)
แต่ละเขตโปเกมอนต่างกันตามธาตุ + พื้นหลัง/ตกแต่งต่างกัน + แสดง % โอกาสเจอแต่ละระดับ

## รันทดสอบบนเครื่อง (Windows)
```powershell
powershell -ExecutionPolicy Bypass -File serve.ps1 -Port 8777
# เปิด http://localhost:8777/
```
> เปิดไฟล์ตรงๆ ด้วย `file://` ก็เล่นได้ แต่ `localStorage` อาจไม่เซฟถาวรบางเบราว์เซอร์ — แนะนำให้เสิร์ฟผ่าน http หรือโฮสต์จริง

## Deploy / ฝังใน MakeWebEasy
อัปโหลด `index.html`, `style.css`, `game.js`, `monsters-data.js` ขึ้นโฮสต์ใดก็ได้ (เช่น GitHub Pages)
แล้วฝังด้วย iframe:
```html
<iframe src="URL-ของเกม" style="width:100%;max-width:560px;height:800px;border:0"></iframe>
```

## ⚠️ หมายเหตุลิขสิทธิ์
ชื่อ/รูป Pokémon เป็นทรัพย์สินทางปัญญาของ Nintendo/Game Freak/The Pokémon Company
เหมาะกับการใช้ส่วนตัว/ทดลอง หากจะเผยแพร่สาธารณะเชิงพาณิชย์ ควรเปลี่ยนไปใช้มอนสเตอร์ต้นฉบับ
โครงสร้าง `monsters-data.js` ออกแบบให้สลับข้อมูล/รูปได้ง่ายในภายหลัง

## เพิ่ม/แก้มอนสเตอร์
แก้ `monsters-data.js` ได้ตรงๆ แต่ละตัวมี:
`id, name, types[], stats{hp,atk,def,spatk,spdef,spd}, rarity, captureRate, evolvesTo, evolveLevel, evolveItem`
หรือรัน `build-data.ps1` ใหม่เพื่อดึงจาก PokeAPI (ปรับ `$MaxId` เพื่อเพิ่ม Gen อื่น)
